Esta es una aplicacion web que sirve para empresas de ventas.

Lo que permite esta aplicacion es lo siguiente:

  -Segun el rol que tengas puedes agregar prodcutos con sus respectivas caracteristicas
  -Crear, editar, borrar, y hasta generar promociones con esos productos
  -Generador de lista de difusion automatica segun las promociones y descuentos que selecciones (te da un mensaje predeterminado listo para copiar y pegar en redes sociales)
  -Generador de Flyers de alta calidad. Tenes para elegir varias plantillas y el producto que vos quieras
